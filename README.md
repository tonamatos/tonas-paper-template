# Template for Tona's academic papers

A PDF gets rendered with every commit for a persistent live version you can share or embed in your website. See [here](https://tonamatos.github.io/tonas-paper-template/main.pdf) for the rendered template.

You can also make a simple `html` landing page and host it on GitHub Pages. [Click here](https://tonamatos.github.io/DeepComputationsNIP/) for an example.

Meant to be used with my style files. See [here](https://github.com/tonamatos/tonas-latex).

## Installation and usage

Clone as a template by clicking "Use as a template," replace with your own files. Don't forget to install and update the submodules by running the following command.

```sh
git submodule update --remote --merge
```
